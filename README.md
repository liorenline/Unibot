# Lost & Found Telegram Bot

## Features

- Users can report lost or found items via an interactive Telegram bot form
- Support request system with direct message forwarding to moderation team
- Menu-based navigation for quick access to main bot functions

## Tech Stack
- Python
- python-telegram-bot
- Telegram Bot API
- Fly.io (deployment)

## Deployment

- The bot is deployed using Fly.io, providing reliable cloud hosting and continuous bot availability.

## Run locally

```bash
git clone https://github.com/liorenline/Unibot.git
cd Unibot
pip install -r requirements.txt
export BOT_TOKEN=your_token
python userbot.py
